# Metropolis Budget Sim (首都圏ライフスタイル設計シミュレーター)

「家賃は手取りの3分の1」という古い定説を捨て、現代の首都圏で「自分を殺さずに」生き抜くための、攻めの家計配分シミュレーターです。

## 🚀 Live Demo
[https://mug-lab-3.github.io/metropolis-budget-sim/](https://mug-lab-3.github.io/metropolis-budget-sim/)

## 📌 コンセプト
現代の首都圏（都心1時間圏内）において、QOL（生活の質）と将来の資産形成を両立させることは「無理ゲー」に近い難易度になっています。このツールは、統計データと独自の「サバイバル理論」に基づき、20代後半のビジネスパーソンが取るべき最適な家計配分を可視化します。

### 特徴
- **動的スライド配分:** 収入増に合わせて、単なる貯金だけでなく「住環境（家賃上限）」や「自己投資」の枠を自動調整。
- **投資バッファ論:** 低収入時は無理な投資をせず、まず「環境」と「経験」に全振りして稼ぐ力を磨くフェーズとして肯定。
- **28万の境界線:** 首都圏で「環境・経験・未来」の三権分立が成立する理想のラインを定義。

## 🛠 使い方
1. **想定月収を入力:** ボーナスを含めた年間手取りを12分割した額を入力してください。
2. **配分の確認:** 独自のアルゴリズムに基づいた「家賃上限」「生活費」「自己投資」「資産形成」の4項目が算出されます。
3. **アラート機能:** 現在の収支バランスが健全かどうか、リアルタイムで判定・アドバイスが表示されます。
4. **THEORYコラム:** 右側のトグルボタンで、この計算式の背景にある設計思想を詳しく読むことができます。

## 📖 設計思想 (The Theory)
- **家賃 = 生産性への投資:** 職住近接や、PC作業が捗るスペースの確保は、最大の自己研鑽である。
- **アンチ・1/3ルール:** 家賃相場が高騰する首都圏で古い定説に縛られると、若者の「今」の経験が奪われる。
- **動的ポートフォリオ:** 収入増の40%を家賃、20%を生活、15%を経験、25%を投資へ。

## ⚖️ 免責事項
本アプリの計算式は、2026年現在の都心部家賃相場および物価上昇率を加味して独自に算出したものです。
